3. typeof null 和typeof NaN 会输出什么
```
    typeof null === 'object'
    typeof NaN === 'number'
    null为object为啥？
    js在底层存储变量的时候，会在变量的机器码的低位1-3位存储其类型信息：
        1. 000 ：对象；
        2. 010 ：浮点数；
        3. 100 ：字符串；
        4. 110 ：布尔值；
        5. 1 ：整数；
        6. 特例：
        （1） null 所有机器码均为 0 
        （2） undefined ：用 −2^30 整数来表示
        typeof 就是通过机器码判断类型，由于null的所有机器码均为0，该机器码和对象一样，因此直接被当作对象来看待，所以通过 typeof 就不能够判断区分对象还有 null 了。
```
